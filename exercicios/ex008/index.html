Nesta frase, temos um termo em ênfase usando a tag <em> ✅

Texte marcado

Podemos criar também um texto marcado usando a tag <mark>.

Texto grande pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado. tag <del>

Texto inserido

Podemos marcar um texto como inserido para dar ênfase que ele foi adicionado depois. tag <ins>

Existe também o sublinhado com a tag <u>

Texto sobrescrito

Para inserir coisas do tipo x20+3 é a tag <sup>

Texto subscrito

Para inserir coisas do tipo H2O é a tag <sub>
